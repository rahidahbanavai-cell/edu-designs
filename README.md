Three interface concepts for the Ghostwriter Content Package tool — a structured bundle that enables AI-native, on-brand content generation at scale.

**[View designs in Figma →](https://www.figma.com/design/kqC2IF3BCbDQnbuaBjtSpM)**

| Concept | Description |
|---|---|
| A · Wizard Flow | Step-by-step guided interface through Input → Process → Output |
| B · Dashboard | All 3 layers visible simultaneously in a 3-column layout |
| C · Split Panel | Configure on the left, live content preview on the right |
